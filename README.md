This is a frontend project where I recreated a GoDaddy-style landing page using HTML, CSS and JavaScript.
The main focus of this project was layout, responsiveness, UI interactions and JavaScript logic.
This project is made only for learning and practice purposes.

TECH USED:
HTML5,
CSS3 (Flexbox, Grid, Media Queries),
JavaScript (DOM manipulation, events, async logic, Intersection Observer and much more),
No frameworks or libraries are used.

FEATURES:
Responsive layout,
Domain search input with basic validation,
Simulated domain availability check,
Loading, success and error messages,
Interactive UI with hover effects and animations,
Dropdown menus in header,
Image hover preview effect,
Signup form UI with validation,
Clean and structured CSS.

WHAT I LEARNED:
Building complex layouts using Flexbox and Grid,
Making large pages responsive,
Writing cleaner CSS for real-world UI,
DOM manipulation and event handling in JavaScript,
Showing dynamic messages (loading, success, error),
Improving UI/UX with small animations.
